What is a Callback?

A callback is a function passed as an argument to another function and is executed after some operation is completed. Callbacks ensure that a function is not executed before a task is finished, but will execute right after the task is completed.

Why Use Callbacks?

  • To handle asynchronous operations like fetching data from an API.
  • To execute code after a specific task is completed.
  • To improve code modularity and reusability.

Examples of Callbacks

1. Simple Callback


<button onclick="executeSimpleCallback()">Run Simple Callback</button>
<p id="simpleCallbackResult"></p>
<script>
    function displayMessage(message) {
        document.getElementById("simpleCallbackResult").textContent = message;
    }

    function executeSimpleCallback() {
        function callback() {
            displayMessage("This is a simple callback function!");
        }
        callback();
    }
</script>

2. Callback with Parameters


<button onclick="calculate(5, 10, displaySum)">Calculate Sum</button>
<p id="sumResult"></p>
<script>
    function calculate(a, b, callback) {
        const result = a + b;
        callback(result);
    }

    function displaySum(sum) {
        document.getElementById("sumResult").textContent = "Sum: " + sum;
    }
</script>

3. Asynchronous Callback


<button onclick="fetchData()">Fetch Data</button>
<p id="dataResult"></p>
<script>
    function fetchData() {
        setTimeout(function() {
            document.getElementById("dataResult").textContent = "Data fetched successfully!";
        }, 2000);
    }
</script>

4. Callback in Event Listener


<button id="eventCallbackButton">Click Me</button>
<p id="eventCallbackResult"></p>
<script>
    document.getElementById("eventCallbackButton").addEventListener("click", function() {
        document.getElementById("eventCallbackResult").textContent = "Button clicked! Callback executed!";
    });
</script>

5. Using Callbacks for Array Processing


<p>Check the console for array processing results.</p>
<script>
    const numbers = [1, 2, 3, 4, 5];

    function processArray(array, callback) {
        for (let i = 0; i < array.length; i++) {
            callback(array[i]);
        }
    }

    processArray(numbers, function(number) {
        console.log("Processing number:", number);
    });
</script>

Advantages of Callbacks

  • Enables asynchronous programming.
  • Makes the code more flexible and reusable.
  • Allows execution of code after a specific task is completed.

Common Use Cases of Callbacks

  • Fetching data from APIs (e.g., using XMLHttpRequest or fetch).
  • Event handling in DOM (e.g., addEventListener).
  • Performing operations on array elements (e.g., forEach, map, filter).
  • Handling asynchronous operations like timers and animations.
