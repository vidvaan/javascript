What is a String in JavaScript?

A string is a sequence of characters used to represent text in JavaScript. Strings are immutable, meaning once they are created, they cannot be changed.

Example:

let greeting = "Hello, World!"; // A string example

JavaScript String Methods

JavaScript provides numerous methods to manipulate and interact with strings. Below are 30 commonly used string methods, along with their descriptions and examples:

Method Description Example Output
charAt() Returns the character at the specified index. let str = "Hello"; str.charAt(1); e
concat() Combines two or more strings into one string. let str = "Hello"; str.concat(", World!"); Hello, World!
includes() Checks if a string contains a specified substring. let str = "Hello"; str.includes("ell"); true
indexOf() Returns the index of the first occurrence of a specified value. let str = "Hello"; str.indexOf("l"); 2
lastIndexOf() Returns the index of the last occurrence of a specified value. let str = "Hello"; str.lastIndexOf("l"); 3
replace() Replaces a specified substring with a new substring. let str = "Hello"; str.replace("e", "a"); Hallo
split() Splits a string into an array of substrings. let str = "Hello, World"; str.split(","); ["Hello", " World"]
toLowerCase() Converts all characters in a string to lowercase. let str = "Hello"; str.toLowerCase(); hello
toUpperCase() Converts all characters in a string to uppercase. let str = "Hello"; str.toUpperCase(); HELLO
trim() Removes whitespace from both ends of a string. let str = " Hello "; str.trim(); Hello
substring() Returns a substring between two specified indices. let str = "Hello"; str.substring(1, 4); ell
slice() Extracts a section of a string and returns it as a new string. let str = "Hello"; str.slice(1, 4); ell
charCodeAt() Returns the Unicode value of the character at the specified index. let str = "Hello"; str.charCodeAt(1); 101
fromCharCode() Returns a string created from the specified sequence of Unicode values. String.fromCharCode(72, 101, 108, 108, 111); Hello
repeat() Returns a new string with a specified number of copies of the string. let str = "Hello"; str.repeat(3); HelloHelloHello
padStart() Pads the string from the start with a specified character. let str = "5"; str.padStart(3, "0"); 005
padEnd() Pads the string from the end with a specified character. let str = "5"; str.padEnd(3, "0"); 500
match() Searches for a match between a string and a regular expression. let str = "Hello"; str.match(/l/g); ["l", "l"]
search() Executes a search for a match between a string and a regular expression. let str = "Hello"; str.search("l"); 2
localeCompare() Compares two strings in the current locale. let str1 = "apple"; let str2 = "banana"; str1.localeCompare(str2); -1
codePointAt() Returns the Unicode code point value of the character at the specified position. let str = "Hello"; str.codePointAt(1); 101
normalize() Returns the Unicode Normalization Form of a string. let str = "e\u0301"; str.normalize(); Ã©
startsWith() Checks if a string starts with a specific substring. let str = "Hello"; str.startsWith("He"); true
endsWith() Checks if a string ends with a specific substring. let str = "Hello"; str.endsWith("lo"); true
trimStart() Removes whitespace from the beginning of a string. let str = " Hello"; str.trimStart(); Hello
trimEnd() Removes whitespace from the end of a string. let str = "Hello "; str.trimEnd(); Hello
