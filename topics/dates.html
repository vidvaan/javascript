What is a Date in JavaScript?

The Date object in JavaScript represents a single moment in time in a platform-independent format. It is used to work with dates and times, including getting the current date, setting custom dates, and performing date calculations.

Example:

let currentDate = new Date(); // Creates a new Date object representing the current date and time

JavaScript Date Methods

The Date object in JavaScript has a variety of methods for manipulating and formatting dates. Below are 15 commonly used methods, along with their descriptions and examples:

Method Description Example Output
getFullYear() Returns the year (4 digits) of the specified date. let date = new Date(); date.getFullYear(); 2024
getMonth() Returns the month (0-11) of the specified date (0 = January, 11 = December). let date = new Date(); date.getMonth(); 11 (for December)
getDate() Returns the day of the month (1-31) of the specified date. let date = new Date(); date.getDate(); 19
getDay() Returns the day of the week (0-6) of the specified date (0 = Sunday, 6 = Saturday). let date = new Date(); date.getDay(); 3 (for Wednesday)
getHours() Returns the hour (0-23) of the specified date. let date = new Date(); date.getHours(); 14
getMinutes() Returns the minutes (0-59) of the specified date. let date = new Date(); date.getMinutes(); 30
getSeconds() Returns the seconds (0-59) of the specified date. let date = new Date(); date.getSeconds(); 45
getMilliseconds() Returns the milliseconds (0-999) of the specified date. let date = new Date(); date.getMilliseconds(); 500
getTime() Returns the number of milliseconds since January 1, 1970 (UTC). let date = new Date(); date.getTime(); 1702992305000
setFullYear() Sets the year of the specified date. let date = new Date(); date.setFullYear(2025); New date with the year 2025
setMonth() Sets the month of the specified date (0-11). let date = new Date(); date.setMonth(5); New date set to June
setDate() Sets the day of the month (1-31) of the specified date. let date = new Date(); date.setDate(10); New date with the 10th day
setHours() Sets the hour (0-23) of the specified date. let date = new Date(); date.setHours(9); New date with the hour set to 9 AM
setMinutes() Sets the minutes (0-59) of the specified date. let date = new Date(); date.setMinutes(45); New date with the minutes set to 45
toDateString() Returns a string representing the date portion of the specified date in a human-readable format. let date = new Date(); date.toDateString(); Wed Dec 19 2024
toISOString() Returns a string representing the date in ISO 8601 format (YYYY-MM-DDTHH:mm:ss.sssZ). let date = new Date(); date.toISOString(); 2024-12-19T09:30:45.500Z
toLocaleDateString() Returns a string representing the date portion in the local format. let date = new Date(); date.toLocaleDateString(); 12/19/2024
toLocaleTimeString() Returns a string representing the time portion in the local format. let date = new Date(); date.toLocaleTimeString(); 09:30:45 AM
toString() Returns a string representing the entire date and time. let date = new Date(); date.toString(); Wed Dec 19 2024 09:30:45 GMT+0000 (Coordinated Universal Time)

Working with Date in JavaScript

JavaScript provides a variety of ways to work with dates. You can create a new date, manipulate it, and format it according to your needs. Use the methods listed above to interact with date objects.
