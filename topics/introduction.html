JavaScript Introduction

JavaScript is a high-level, interpreted programming language primarily used for building interactive and dynamic web applications. It was created by Brendan Eich in 1995 and has since become one of the core technologies of web development, alongside HTML and CSS. JavaScript enables you to add interactivity, control multimedia, animate elements, and much more on a webpage.

Uses of JavaScript

  • Web Development: JavaScript is essential for client-side scripting. It is used to make websites interactive, responsive, and dynamic. It can manipulate the DOM (Document Object Model) of HTML pages, enabling the creation of dynamic interfaces.
  • Server-side Development: With the advent of Node.js, JavaScript can also be used on the server-side to create scalable network applications.
  • Game Development: JavaScript can be used to create browser-based games, and libraries like Phaser help in game development.
  • Mobile App Development: JavaScript can be used to develop mobile applications using frameworks like React Native, Ionic, and more.
  • Data Visualization: JavaScript can visualize data in web applications through libraries like D3.js.
  • Automation: JavaScript can automate tasks via scripting, such as web scraping or automating user interactions.

Programming Language vs Scripting Language

Programming Language: A programming language is a formal set of instructions that can be used to produce a variety of outputs, such as creating software or algorithms. Examples: Java, C++, Python, etc. A programming language can be compiled (translated into machine code) or interpreted (executed by an interpreter).

Scripting Language: A scripting language is a subset of programming languages that are typically interpreted (not compiled) and used to automate specific tasks or control the behavior of other software applications. Examples: JavaScript, Python (also considered a scripting language), PHP, etc.

Key Difference: While both can be used for general-purpose programming, scripting languages are often lightweight, interpreted, and used for automating tasks, typically within a host environment (e.g., a web browser or server).

Creating Your First JavaScript Application

Let's create a simple JavaScript application that displays a message in the browser and through Node.js.

1. Run JavaScript with a Browser

Create an HTML file with embedded JavaScript:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>First JavaScript App</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <button onclick="displayMessage()">Click me</button>

    <script>
        function displayMessage() {
            alert("Hello, this is your first JS application!");
        }
    </script>
</body>
</html>

Instructions:

  • Save the file as index.html.
  • Open the index.html file in a web browser.
  • The page will display a "Click me" button. When clicked, a JavaScript alert will appear.

2. Run JavaScript with Node.js and VSCode Terminal

Follow these steps to run JavaScript using Node.js in the VSCode terminal:


<script>
console.log("Hello, World!");
</script>

Instructions:

  • Install Node.js if not already done from here.
  • Open VSCode and create a new file, say app.js.
  • In the terminal, navigate to the folder where app.js is located.
  • Run the following command:
    • node app.js
  • The terminal will display Hello, World!.

Generate HTML with JavaScript

You can dynamically generate HTML content using JavaScript. Here's an example of how to create HTML elements using JavaScript and display them within a webpage.

Example: Dynamic HTML Generation


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Dynamic HTML Generation</title>
</head>
<body>
    <h1>Generated Content</h1>
    <div id="dynamicContent"></div>

    <script>
        const content = `
            <p>This content was generated dynamically using JavaScript!</p>
            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
                <li>Item 3</li>
            </ul>
        `;
        document.getElementById("dynamicContent").innerHTML = content;
    </script>
</body>
</html>

Steps:

  • Save the file as dynamic.html.
  • Open it in a browser.
  • The content within the <div id="dynamicContent"> will be dynamically generated by JavaScript.
