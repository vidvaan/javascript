JavaScript Objects

In JavaScript, an object is a collection of properties and methods. A property is a key-value pair, where the key is a string (or symbol), and the value can be any data type, including another object or function. Objects allow you to store and organize data in a structured way.

1. Creating Objects in JavaScript

There are several ways to create objects in JavaScript:

1.1 Object Literal Notation

The most common way to create an object is by using object literal notation.

Syntax:

                
const objectName = {
    key1: value1,
    key2: value2,
    // More properties
};

Example 1:

                
const person = {
    name: "Alice",
    age: 30,
    greet: function() {
        console.log("Hello " + this.name);
    }
};
person.greet(); // Hello Alice

This example shows a simple object `person` with three properties: `name`, `age`, and a method `greet`. The method uses `this` to refer to the object itself and access its properties.

1.2 Using the `new Object()` Syntax

Another way to create an object is by using the `new Object()` constructor.

Syntax:

                
const objectName = new Object();
objectName.key1 = value1;
objectName.key2 = value2;

Example 2:

                
const person = new Object();
person.name = "Bob";
person.age = 25;
person.greet = function() {
    console.log("Hello " + this.name);
};
person.greet(); // Hello Bob

Here, we create an empty object using the `new Object()` constructor and then add properties and methods to it.

1.3 Using a Constructor Function

You can also create objects using constructor functions, which allow you to create multiple instances of similar objects.

Syntax:

                
function Person(name, age) {
    this.name = name;
    this.age = age;
    this.greet = function() {
        console.log("Hello " + this.name);
    };
}
const person1 = new Person("Charlie", 35);
person1.greet(); // Hello Charlie

Example 3:

                
function Person(name, age) {
    this.name = name;
    this.age = age;
    this.greet = function() {
        console.log("Hello " + this.name);
    };
}
const person1 = new Person("Charlie", 35);
const person2 = new Person("David", 40);
person1.greet(); // Hello Charlie
person2.greet(); // Hello David

The constructor function `Person` is used to create multiple objects (`person1`, `person2`) with their own properties and methods.

1.4 Using `Object.create()` Method

JavaScript also provides the `Object.create()` method, which is used to create a new object using an existing object as a prototype.

Syntax:

                
const newObject = Object.create(prototypeObject);

Example 4:

                
const personPrototype = {
    greet: function() {
        console.log("Hello " + this.name);
    }
};
const person1 = Object.create(personPrototype);
person1.name = "Eve";
person1.greet(); // Hello Eve

In this example, `personPrototype` serves as the prototype for the new object `person1`. The method `greet` is inherited from the prototype.

2. Accessing and Modifying Object Properties

You can access and modify the properties of objects using either dot notation or bracket notation.

2.1 Dot Notation

                
person.name = "Alice";
console.log(person.name); // Alice

Dot notation is the most common way to access and modify object properties.

2.2 Bracket Notation

                
person["age"] = 30;
console.log(person["age"]); // 30

Bracket notation allows you to access or modify properties dynamically using a string or variable.

3. Object Methods

Objects can also have methods, which are functions associated with an object.

Example:

                
const car = {
    make: "Toyota",
    model: "Corolla",
    year: 2021,
    displayInfo: function() {
        console.log(`Car: ${this.year} ${this.make} ${this.model}`);
    }
};
car.displayInfo(); // Car: 2021 Toyota Corolla

In this example, the `car` object has a method `displayInfo` which displays the car's information using the `this` keyword.

4. Object Destructuring

Object destructuring allows you to extract properties from an object into variables.

Syntax:

                
const { property1, property2 } = objectName;

Example:

                
const person = {
    name: "Alice",
    age: 30,
    country: "USA"
};
const { name, age } = person;
console.log(name); // Alice
console.log(age);  // 30

Here, we extract the `name` and `age` properties from the `person` object and assign them to variables using destructuring.

5. Adding and Removing Properties

You can add and remove properties from an object at any time.

Example: Adding a Property

                
person.city = "New York";
console.log(person.city); // New York

Example: Removing a Property

                
delete person.city;
console.log(person.city); // undefined

In these examples, we add the `city` property to the `person` object and then remove it using the `delete` keyword.
