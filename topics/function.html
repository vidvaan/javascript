JavaScript Functions

Functions in JavaScript are blocks of code that can be defined once and executed whenever needed. Functions help make code more modular, reusable, and easier to maintain.

1. Function Declaration

A function declaration defines a function that can be called anywhere in the code after it's declared.

Syntax:

function functionName(parameters) {
    // function body
    return value; // optional
}

Example 1:

function greet(name) {
    return "Hello " + name;
}
console.log(greet("Alice")); // Hello Alice

The function greet takes a parameter name and returns a greeting message. When called with "Alice", it outputs "Hello Alice".

Example 2:

function multiply(a, b) {
    return a * b;
}
console.log(multiply(2, 3)); // 6

The function multiply takes two parameters a and b and returns their product. When called with 2 and 3, it outputs 6.

2. Function Expression (Anonymous Function)

A function expression is a function assigned to a variable. It can be anonymous (without a name) and is often used for callbacks or as a return value.

Syntax:

const functionName = function(parameters) {
    // function body
    return value; // optional
};

Example 1:

const sayHello = function(name) {
    return "Hello " + name;
};
console.log(sayHello("Bob")); // Hello Bob

An anonymous function is assigned to sayHello. It outputs "Hello Bob" when called with "Bob".

Example 2:

const square = function(x) {
    return x * x;
};
console.log(square(4)); // 16

The function square calculates the square of the input. When called with 4, it returns 16.

3. Arrow Functions

Arrow functions offer a concise way to write functions. They are especially useful for callbacks or in functional programming.

Syntax:

const functionName = (parameters) => {
    // function body
    return value; // optional
};

Example 1:

const add = (a, b) => a + b;
console.log(add(3, 4)); // 7

The add function returns the sum of two numbers. When called with 3 and 4, it outputs 7.

Example 2:

const greetUser = name => `Hello, ${name}!`;
console.log(greetUser("Charlie")); // Hello, Charlie!

The greetUser function uses template literals to return a greeting. It outputs "Hello, Charlie!" when called with "Charlie".

Example 3 (without curly braces):

const double = x => x * 2;
console.log(double(5)); // 10

If the function body has a single expression, curly braces and return are optional. The double function multiplies input by 2.

Example 4 (with multiple parameters):

const multiplyNumbers = (a, b) => {
    return a * b;
};
console.log(multiplyNumbers(2, 3)); // 6

The multiplyNumbers function returns the product of two numbers.

4. Functions with Default Parameters

Default parameters are used when no argument is passed for a parameter.

Syntax:

function functionName(param = defaultValue) {
    // function body
}

Example:

function greet(name = "Guest") {
    return "Hello " + name;
}
console.log(greet()); // Hello Guest
console.log(greet("Alice")); // Hello Alice

The greet function defaults name to "Guest" if no argument is provided.

5. Functions as Arguments (Callback Functions)

Functions can be passed as arguments to other functions, making them versatile for callbacks and event handling.

Syntax:

function mainFunction(callback) {
    callback();
}
mainFunction(() => console.log("Callback function executed"));

Example:

function greetUser(name, callback) {
    console.log("Hello " + name);
    callback();
}

greetUser("Alice", function() {
    console.log("Goodbye Alice!");
});

The greetUser function takes a name and a callback, greeting the user and then executing the callback.
