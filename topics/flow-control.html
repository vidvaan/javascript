JavaScript Control Flow

Control flow in JavaScript determines the execution order of code. It includes conditional statements, loops, and jump statements for effective flow management.

1. Conditional Statements

Conditional statements enable decision-making by executing specific code blocks based on conditions.

if, else, else if

These statements control the execution flow based on the truthiness of conditions.

Syntax:


if (condition) {
    // executes if condition is true
} else {
    // executes if condition is false
}

Example:


let age = 20;

if (age > 18) {
    console.log("Adult");
} else {
    console.log("Not an adult");
}

else if

Tests multiple conditions sequentially until one evaluates to true.

Syntax:


if (condition1) {
    // executes if condition1 is true
} else if (condition2) {
    // executes if condition2 is true
} else {
    // executes if none of the above conditions are true
}

Example:


let age = 25;

if (age > 60) {
    console.log("Senior");
} else if (age > 18) {
    console.log("Adult");
} else {
    console.log("Not an adult");
}

switch

Performs multiple checks on the same value for better readability.

Syntax:


switch (expression) {
    case value1:
        // executes if expression === value1
        break;
    case value2:
        // executes if expression === value2
        break;
    default:
        // executes if no case matches
}

Example:


let day = 2;
let dayName;

switch (day) {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    case 3:
        dayName = "Wednesday";
        break;
    default:
        dayName = "Invalid day";
        break;
}

console.log(dayName); // Outputs: Tuesday

2. Loops

Loops repeat code execution based on conditions.

for Loop

Best for fixed iteration count.

Syntax:


for (initialization; condition; increment) {
    // executes while condition is true
}

Example:


for (let i = 0; i < 5; i++) {
    console.log(i); // Outputs: 0 to 4
}

while Loop

Repeats as long as the condition remains true.

Syntax:


while (condition) {
    // executes while condition is true
}

Example:


let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

do...while Loop

Executes at least once before condition check.

Syntax:


do {
    // executes once and then checks condition
} while (condition);

Example:


let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

3. Jump Statements

Alter the flow of execution in loops and functions.

break

Exits the loop or switch prematurely.

Example:


for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i); // Outputs: 0 to 4
}

continue

Skips the current iteration and proceeds to the next.

Example:


for (let i = 0; i < 5; i++) {
    if (i === 3) continue;
    console.log(i); // Outputs: 0, 1, 2, 4
}

return

Stops function execution and optionally returns a value.

Example:


function checkAge(age) {
    if (age < 18) return "Not an adult";
    return "Adult";
}

console.log(checkAge(20)); // Outputs: Adult
