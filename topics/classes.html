JavaScript Classes and Constructors

Classes in JavaScript are templates for creating objects. They encapsulate data and methods related to the objects. The constructor method is a special method used for initializing an objectâ€™s properties when it is created.

1. Basic Class and Constructor

A simple class with a constructor that initializes object properties.


class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return `Hello, my name is ${this.name} and I am ${this.age} years old.`;
    }
}

const person1 = new Person("Alice", 25);
console.log(person1.greet()); // Outputs: Hello, my name is Alice and I am 25 years old.

2. Default Values in Constructor

Using default values for constructor parameters when no value is provided.


class Car {
    constructor(brand = "Generic", model = "Model X") {
        this.brand = brand;
        this.model = model;
    }

    describe() {
        return `This car is a ${this.brand} ${this.model}.`;
    }
}

const car1 = new Car();
const car2 = new Car("Tesla", "Model S");
console.log(car1.describe()); // Outputs: This car is a Generic Model X.
console.log(car2.describe()); // Outputs: This car is a Tesla Model S.

3. Method Overloading Using Constructors

Simulating method overloading by checking parameters inside the constructor.


class Rectangle {
    constructor(width, height) {
        if (height === undefined) {
            this.width = width;
            this.height = width; // Create a square if only one parameter is passed
        } else {
            this.width = width;
            this.height = height;
        }
    }

    area() {
        return this.width * this.height;
    }
}

const square = new Rectangle(5);
const rectangle = new Rectangle(4, 6);
console.log(square.area()); // Outputs: 25
console.log(rectangle.area()); // Outputs: 24

4. Inheritance and Constructors

Extending a class and using the super keyword to call the parent constructor.


class Animal {
    constructor(name) {
        this.name = name;
    }

    speak() {
        return `${this.name} makes a noise.`;
    }
}

class Dog extends Animal {
    constructor(name, breed) {
        super(name); // Call the parent constructor
        this.breed = breed;
    }

    speak() {
        return `${this.name}, the ${this.breed}, barks.`;
    }
}

const dog = new Dog("Buddy", "Golden Retriever");
console.log(dog.speak()); // Outputs: Buddy, the Golden Retriever, barks.

5. Static Methods in Classes

Static methods are called on the class itself, not on an instance of the class.


class MathUtil {
    static add(a, b) {
        return a + b;
    }

    static multiply(a, b) {
        return a * b;
    }
}

console.log(MathUtil.add(5, 3)); // Outputs: 8
console.log(MathUtil.multiply(4, 2)); // Outputs: 8
