What are JavaScript Events?

Events in JavaScript are actions or occurrences that happen in the browser, such as when a user clicks on an element, submits a form, or moves the mouse over an element. JavaScript allows us to handle these events through event listeners, which respond when the event occurs.

AddingEvents Examples

Here are some examples of adding events in different ways:

Inline Event Handling:

<button onclick="alert('Button clicked!')">Click Me</button>

Event Handler Property:


<button id="btn1">Click Me</button>
<script>
document.getElementById('btn1').onclick = function() {
    alert('Button clicked using event handler property!');
};
</script>

Event Listener:


<button id="btn2">Click Me</button>
<script>
document.getElementById('btn2').addEventListener('click', function() {
    alert('Button clicked using event listener!');
});
</script>

Ways to Add Events in JavaScript

There are three common ways to add events to elements in JavaScript:

  • Inline Event Handling: Adding event handler directly in the HTML element using the onEvent attribute (e.g., onclick, onmouseover).
  • Event Handler Property: Assigning an event handler function to an element's event property (e.g., element.onclick = function()).
  • Event Listener: Using addEventListener() to attach an event handler to an element, which allows multiple handlers for the same event.

Common JavaScript Events

Below are a couple of common JavaScript events:

Click Event

<button id="clickBtn">Click Me</button>
<script>
document.getElementById('clickBtn').addEventListener('click', function() {
    alert('Button clicked!');
});
</script>

Mouseover Event

<div id="hoverDiv">Hover over me!</div>
<script>
document.getElementById('hoverDiv').addEventListener('mouseover', function() {
    alert('Mouse over element!');
});
</script>

Keydown Event

<script>
document.addEventListener('keydown', function(event) {
    console.log('Key pressed:', event.key);
});
</script>

Submit Event

<form id="submitForm">
    <input type="text" placeholder="Type something" required>
    <button type="submit">Submit</button>
</form>
<script>
document.getElementById('submitForm').addEventListener('submit', function(event) {
    event.preventDefault();
    alert('Form submitted!');
});
</script>

Resize Event

<script>
window.addEventListener('resize', function() {
    console.log('Window resized!');
});
</script>

Scroll Event

<script>
window.addEventListener('scroll', function() {
    console.log('Page scrolled!');
});</script>

Focus Event

<input type="text" id="focusInput" placeholder="Focus on me">
<script>
document.getElementById('focusInput').addEventListener('focus', function() {
    console.log('Input field focused!');
});
</script>

Blur Event


<input type="text" id="blurInput" placeholder="Blur from me">
<script>
document.getElementById('blurInput').addEventListener('blur', function() {
    console.log('Input field lost focus!');
});
</script>

Change Event


<select id="changeSelect">
    <option value="Option 1">Option 1</option>
    <option value="Option 2">Option 2</option>
    <option value="Option 3">Option 3</option>
</select>
<script>
document.getElementById('changeSelect').addEventListener('change', function() {
    console.log('Value changed!');
});
</script>

Drag Event


<div id="dragDiv" draggable="true">Drag me!</div>
<script>
document.getElementById('dragDiv').addEventListener('drag', function() {
    console.log('Element is being dragged!');
});
</script>

Drop Event


<div id="dropDiv">Drop here!</div>
<script>
document.getElementById('dropDiv').addEventListener('drop', function() {
    console.log('Element dropped!');
});
</script>

Input Event


<input type="text" id="inputField" placeholder="Type something">
<script>
document.getElementById('inputField').addEventListener('input', function() {
    console.log('Input typed!');
});
</script>

Load Event


<script>
window.addEventListener('load', function() {
    console.log('Page loaded!');
});
</script>

Unload Event


<script>
window.addEventListener('unload', function() {
    console.log('Page unloaded!');
});
</script>

Double-click Event


<button id="dblClickBtn">Double-click Me</button>
<script>
document.getElementById('dblClickBtn').addEventListener('dblclick', function() {
    alert('Element double-clicked!');
});
</script>

Context Menu Event


<div id="contextDiv">Right-click on me!</div>
<script>
document.getElementById('contextDiv').addEventListener('contextmenu', function(event) {
    event.preventDefault();
    alert('Right-click disabled!');
});
</script>

Mouseout Event


<div id="mouseoutDiv">Move your mouse out of me!</div>
<script>
document.getElementById('mouseoutDiv').addEventListener('mouseout', function() {
    alert('Mouse out of element!');
});
</script>

Mousemove Event


<div id="mousemoveDiv">Move your mouse over me!</div>
<script>
document.getElementById('mousemoveDiv').addEventListener('mousemove', function() {
    console.log('Mouse is moving over the element!');
});
</script>

Wheel Event


<script>
window.addEventListener('wheel', function(event) {
    console.log('Mouse wheel scrolled!', event.deltaY);
});
</script>

Error Event


<script>
window.addEventListener('error', function(event) {
    console.error('Error occurred:', event.message);
});
</script>

Beforeunload Event


<script>
window.addEventListener('beforeunload', function(event) {
    event.preventDefault();
    event.returnValue = '';
});
</script>

JavaScript Events Table

Event Description
click Triggered when an element is clicked.
mouseover Triggered when the mouse pointer enters an element.
keydown Triggered when a key is pressed down.
submit Triggered when a form is submitted.
resize Triggered when the window is resized.
scroll Triggered when the document view is scrolled.
focus Triggered when an element gains focus (e.g., an input field).
blur Triggered when an element loses focus.
change Triggered when the value of an input, select, or textarea element changes.
drag Triggered when an element is being dragged.
drop Triggered when a dragged element is dropped onto a valid target.
input Triggered when the value of an input field changes (e.g., typing).
load Triggered when the page or an element (like an image) is fully loaded.
unload Triggered when the page is unloaded.
dblclick Triggered when an element is double-clicked.
contextmenu Triggered when a user right-clicks on an element.
mouseout Triggered when the mouse pointer leaves an element.
mousemove Triggered when the mouse pointer moves over an element.
wheel Triggered when the mouse wheel is scrolled.
error Triggered when an error occurs (e.g., in loading an image).
beforeunload Triggered before the user leaves the page or reloads it.
