JavaScript Operators

In JavaScript, operators are special symbols or keywords that are used to perform operations on values. These operations could be arithmetic, logical, comparison, or assignment-based. Below is a detailed explanation of various types of operators in JavaScript.

1. Arithmetic Operators

Arithmetic operators perform mathematical operations on numbers.

  • + (Addition): Adds two operands.
  • - (Subtraction): Subtracts the second operand from the first.
  • * (Multiplication): Multiplies two operands.
  • / (Division): Divides the first operand by the second.
  • % (Modulus): Returns the remainder when the first operand is divided by the second.

Examples:


let a = 10;
let b = 3;

console.log(a + b); // Outputs: 13
console.log(a - b); // Outputs: 7
console.log(a * b); // Outputs: 30
console.log(a / b); // Outputs: 3.3333
console.log(a % b); // Outputs: 1 (remainder)

2. Comparison Operators

Comparison operators are used to compare two values and return a boolean value (true or false).

  • == (Equal to): Compares two values for equality, ignoring the data type.
  • === (Strict equal to): Compares two values for equality, including the data type.
  • != (Not equal to): Compares two values to check if they are not equal, ignoring the data type.
  • !== (Strict not equal to): Compares two values to check if they are not equal, including the data type.
  • > (Greater than): Checks if the left operand is greater than the right operand.
  • < (Less than): Checks if the left operand is less than the right operand.
  • >= (Greater than or equal to): Checks if the left operand is greater than or equal to the right operand.
  • <= (Less than or equal to): Checks if the left operand is less than or equal to the right operand.

Examples:


let x = 10;
let y = 5;
let z = "10";

console.log(x == y);  // Outputs: false
console.log(x == z);  // Outputs: true (type is ignored)
console.log(x === z); // Outputs: false (type is considered)

console.log(x > y);   // Outputs: true
console.log(x < y);   // Outputs: false
console.log(x >= y);  // Outputs: true
console.log(x <= y);  // Outputs: false

3. Logical Operators

Logical operators are used to combine multiple boolean expressions or values.

  • && (Logical AND): Returns true if both operands are true.
  • || (Logical OR): Returns true if at least one operand is true.
  • ! (Logical NOT): Reverses the logical state of its operand (true becomes false, and false becomes true).

Examples:


let isTrue = true;
let isFalse = false;

console.log(isTrue && isFalse);  // Outputs: false
console.log(isTrue || isFalse);  // Outputs: true
console.log(!isTrue);            // Outputs: false

4. Assignment Operators

Assignment operators are used to assign values to variables.

  • = (Assignment): Assigns the right operandâ€™s value to the left operand.
  • += (Add and assign): Adds the right operand to the left operand and assigns the result to the left operand.
  • -= (Subtract and assign): Subtracts the right operand from the left operand and assigns the result to the left operand.
  • *= (Multiply and assign): Multiplies the left operand by the right operand and assigns the result to the left operand.
  • /= (Divide and assign): Divides the left operand by the right operand and assigns the result to the left operand.

Examples:


let a = 5;

a += 3;  // a = a + 3 => 8
console.log(a); // Outputs: 8

a -= 2;  // a = a - 2 => 6
console.log(a); // Outputs: 6

a *= 4;  // a = a * 4 => 24
console.log(a); // Outputs: 24

a /= 6;  // a = a / 6 => 4
console.log(a); // Outputs: 4

5. Unary Operators

Unary operators operate on a single operand and perform operations like increment, decrement, and negation.

  • ++ (Increment): Increases the operand by 1.
  • -- (Decrement): Decreases the operand by 1.
  • typeof (Type of): Returns the data type of the operand.
  • delete (Delete): Deletes a property from an object.
  • void (Void): Evaluates an expression without returning a value.

Examples:


let x = 10;
console.log(x++);  // Outputs: 10 (post-increment)
console.log(x);    // Outputs: 11

let y = 5;
console.log(--y);  // Outputs: 4 (pre-decrement)

console.log(typeof x);  // Outputs: number

let person = { name: "John", age: 30 };
delete person.age;     // Deletes the 'age' property
console.log(person);    // Outputs: { name: "John" }

console.log(void(0));  // Outputs: undefined

6. Ternary Operator

The ternary operator is a shorthand for an if-else statement.

  • condition ? value_if_true : value_if_false

Example:


let age = 18;
let canVote = (age >= 18) ? "Yes" : "No";
console.log(canVote);  // Outputs: Yes

7. Bitwise Operators

Bitwise operators work on bits and perform bit-by-bit operations.

  • & (Bitwise AND): Performs AND operation on bits.
  • | (Bitwise OR): Performs OR operation on bits.
  • ^ (Bitwise XOR): Performs XOR operation on bits.
  • ~ (Bitwise NOT): Inverts the bits.
  • << (Bitwise left shift): Shifts bits to the left.
  • >> (Bitwise right shift): Shifts bits to the right.

Example:


let a = 5;      // 0101
let b = 3;      // 0011

console.log(a & b);   // Outputs: 1 (0101 & 0011 = 0001)
console.log(a | b);   // Outputs: 7 (0101 | 0011 = 0111)
console.log(a ^ b);   // Outputs: 6 (0101 ^ 0011 = 0110)
console.log(~a);      // Outputs: -6 (inverts bits)
console.log(a << 1);  // Outputs: 10 (shifts left)
console.log(a >> 1);  // Outputs: 2 (shifts right)

8. Type Operators

Type operators are used to check the type of a value.

  • typeof (Type of): Returns the data type of the operand.
  • instanceof (Instance of): Checks if an object is an instance of a particular class or prototype.

Examples:


let num = 10;
console.log(typeof num);  // Outputs: number

let person = { name: "John" };
console.log(person instanceof Object);  // Outputs: true
