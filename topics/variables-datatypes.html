JavaScript Variables and Data Types

JavaScript has three main ways to declare variables and uses various data types for storing values. Understanding how to declare variables and the different data types is essential for writing efficient code.

1. Declaring Variables in JavaScript

In JavaScript, variables can be declared using var, let, and const. These keywords determine the scope and reassignability of the variable.

var

var is function-scoped, meaning it is accessible within the function in which it is declared or globally if declared outside of a function. It can be re-declared and updated.


var x = 10;
var x = 20; // Re-declaring and updating the value
console.log(x); // Outputs 20

let

let is block-scoped, meaning it is only accessible within the block (enclosed by curly braces) in which it is declared. It can be updated but not re-declared within the same block.


let y = 30;
y = 40; // Updating the value
console.log(y); // Outputs 40
// let y = 50; // Error: Cannot redeclare block-scoped variable 'y'

const

const is also block-scoped like let, but its value cannot be updated or re-declared. Once assigned, the value is constant.


const z = 50;
// z = 60; // Error: Assignment to constant variable.
console.log(z); // Outputs 50

2. Data Types in JavaScript

JavaScript has two main categories of data types: Primitive and Reference types.

Primitive Data Types

Primitive types represent single values. They include:

  • String: Represents text.
  • Number: Represents numeric values, both integers and floating-point numbers.
  • Boolean: Represents a true or false value.
  • Undefined: A variable that is declared but not assigned a value has the value undefined.
  • Null: Represents the intentional absence of a value or object.
  • Symbol: Represents a unique identifier, often used for object property keys.
  • BigInt: Used to represent large integers that cannot be represented by the Number type.

Example of Primitive Data Types:


let str = "Hello, World!";    // String
let num = 42;                // Number
let isTrue = true;           // Boolean
let undefinedValue;          // Undefined
let emptyValue = null;       // Null
let uniqueSymbol = Symbol("id"); // Symbol
let bigIntValue = 12345678901234567890n; // BigInt

Reference Data Types

Reference types are more complex and are stored by reference. They include:

  • Object: A collection of key-value pairs. It can store multiple values of different data types.
  • Array: A list-like object, which is an ordered collection of values (can be of any data type).
  • Function: A block of code designed to perform a specific task.

Example of Reference Data Types:


    // Object Example
    let person = {
        name: "John",
        age: 30,
        isStudent: false
    };

    // Array Example
    let numbers = [1, 2, 3, 4, 5];

    // Function Example
    function greet(name) {
        return "Hello, " + name;
    }
    console.log(greet("Alice")); // Outputs: Hello, Alice
