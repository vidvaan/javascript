ES6 Features and Concepts

ES6 (ECMAScript 2015) introduced several new features and enhancements to JavaScript, making the language more powerful and easier to work with. This document explores each feature in detail with examples.

1. Let and Const

ES6 introduced let and const for variable declarations. These keywords provide block scope and help avoid issues caused by the function-scoped var.

let

let allows you to declare variables that are limited in scope to the block in which they are defined.


let count = 10;
if (true) {
    let count = 20; // Block scoped
    console.log(count); // Outputs 20
}
console.log(count); // Outputs 10

const

const is used to declare variables whose values cannot be reassigned.


const PI = 3.14;
// PI = 3.1415; // Error: Assignment to constant variable.
console.log(PI); // Outputs 3.14

2. Arrow Functions

Arrow functions provide a concise syntax for writing functions and have a lexical this value.


// Traditional function
function add(a, b) {
    return a + b;
}

// Arrow function
const add = (a, b) => a + b;
console.log(add(5, 3)); // Outputs 8

3. Template Literals

Template literals allow for easier string interpolation and multi-line strings.


const name = "Alice";
const greeting = `Hello, ${name}!`;
console.log(greeting); // Outputs: Hello, Alice!

const multiline = `This is a
multi-line string.`;
console.log(multiline);

4. Destructuring

Destructuring allows you to unpack values from arrays or properties from objects into distinct variables.

Array Destructuring


const numbers = [1, 2, 3];
const [first, second] = numbers;
console.log(first, second); // Outputs 1, 2

Object Destructuring


const person = { name: "Alice", age: 25 };
const { name, age } = person;
console.log(name, age); // Outputs Alice, 25

5. Default Parameters

Functions can have default values for parameters if no value is provided.


function greet(name = "Guest") {
    return `Hello, ${name}!`;
}
console.log(greet()); // Outputs: Hello, Guest!
console.log(greet("Alice")); // Outputs: Hello, Alice!

6. Spread and Rest Operators

Spread Operator

The spread operator (...) allows you to expand an array or object.


const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];
console.log(arr2); // Outputs [1, 2, 3, 4, 5]

Rest Operator

The rest operator collects all remaining elements into an array.


function sum(...numbers) {
    return numbers.reduce((acc, num) => acc + num, 0);
}
console.log(sum(1, 2, 3)); // Outputs 6

7. Classes

ES6 introduced classes for easier and more structured object-oriented programming in JavaScript.


class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return `Hello, my name is ${this.name}.`;
    }
}
const alice = new Person("Alice", 25);
console.log(alice.greet()); // Outputs: Hello, my name is Alice.

8. Modules

ES6 modules allow you to export and import code between files.

Exporting


// math.js
export const add = (a, b) => a + b;
export const subtract = (a, b) => a - b;

Importing


// app.js
import { add, subtract } from './math.js';
console.log(add(5, 3)); // Outputs 8
