What is an Array in JavaScript?

An array is a special variable in JavaScript that can hold more than one value at a time. It is an ordered collection of elements that can store data of various types, including numbers, strings, objects, and even other arrays.

Arrays are created using square brackets [] or the Array constructor.

Example:

let numbers = [1, 2, 3, 4]; // Array of numbers
let fruits = ["apple", "banana", "cherry"]; // Array of strings

What is Array Iteration?

Array iteration refers to the process of accessing each element of an array, one by one, to perform operations like logging, modifying, or filtering elements. JavaScript provides several ways to iterate through arrays, ranging from simple loops to advanced iteration methods like forEach and map.

Array Iteration Methods with Examples

1. Using for Loop

The for loop iterates through the array using an index.


let arr = [1, 2, 3];
for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}
// Output:
// 1
// 2
// 3

2. Using for...of Loop

The for...of loop iterates through the values of the array.


let arr = [1, 2, 3];
for (let value of arr) {
    console.log(value);
}
// Output:
// 1
// 2
// 3

3. Using forEach()

The forEach() method executes a function once for each array element.


let arr = [1, 2, 3];
arr.forEach(value => console.log(value));
// Output:
// 1
// 2
// 3

4. Using map()

The map() method creates a new array by applying a function to each element.


let arr = [1, 2, 3];
let doubled = arr.map(value => value * 2);
console.log(doubled);
// Output:
// [2, 4, 6]

5. Using filter()

The filter() method creates a new array containing elements that pass a condition.


let arr = [1, 2, 3, 4];
let even = arr.filter(value => value % 2 === 0);
console.log(even);
// Output:
// [2, 4]

6. Using reduce()

The reduce() method reduces the array to a single value.


let arr = [1, 2, 3];
let sum = arr.reduce((acc, value) => acc + value, 0);
console.log(sum);
// Output:
// 6

7. Using while Loop

The while loop iterates through the array while a condition is true.


let arr = [1, 2, 3];
let i = 0;
while (i < arr.length) {
    console.log(arr[i]);
    i++;
}
// Output:
// 1
// 2
// 3

8. Using do...while Loop

The do...while loop executes at least once and continues while a condition is true.


let arr = [1, 2, 3];
let i = 0;
do {
    console.log(arr[i]);
    i++;
} while (i < arr.length);
// Output:
// 1
// 2
// 3

9. Using keys()

The keys() method returns an iterator for the keys (indexes).


let arr = [1, 2, 3];
for (let key of arr.keys()) {
    console.log(key);
}
// Output:
// 0
// 1
// 2

10. Using values()

The values() method returns an iterator for the values.


let arr = [1, 2, 3];
for (let value of arr.values()) {
    console.log(value);
}
// Output:
// 1
// 2
// 3

11. Using entries()

The entries() method returns an iterator for key-value pairs.


let arr = [1, 2, 3];
for (let [index, value] of arr.entries()) {
    console.log(index, value);
}
// Output:
// 0 1
// 1 2
// 2 3
