JavaScript Modules

Modules are a way to organize and reuse JavaScript code. They allow developers to export functions, objects, or values from one file and import them into another. This modular approach makes code more maintainable and scalable.

1. Exporting and Importing

JavaScript modules use the export and import keywords to share and use code across different files.

Example: Named Export and Import

In this example, we export a function and a constant from one file and import them into another:

// math.js
export const PI = 3.14;
export function add(a, b) {
    return a + b;
}

// main.js
import { PI, add } from './math.js';
console.log(PI); // Outputs: 3.14
console.log(add(2, 3)); // Outputs: 5

2. Default Export and Import

A module can have a default export, which is imported without curly braces. Useful for exporting a single main value or function.

Example: Default Export

// greet.js
export default function greet(name) {
    return `Hello, ${name}!`;
}

// main.js
import greet from './greet.js';
console.log(greet('Alice')); // Outputs: Hello, Alice!

3. Importing Everything

You can import all the exports from a module using * as an alias.

Example: Import All

// utils.js
export function square(x) {
    return x * x;
}
export function cube(x) {
    return x * x * x;
}

// main.js
import * as Utils from './utils.js';
console.log(Utils.square(3)); // Outputs: 9
console.log(Utils.cube(3));   // Outputs: 27

4. Re-exporting Modules

Modules can re-export exports from another module, creating a chain of exports.

Example: Re-exporting

// constants.js
export const PI = 3.14;
export const E = 2.71;

// math.js
export * from './constants.js';

// main.js
import { PI, E } from './math.js';
console.log(PI); // Outputs: 3.14
console.log(E);  // Outputs: 2.71

5. Dynamic Imports

Dynamic imports allow you to load modules at runtime, which can improve performance by deferring the loading of unused modules.

Example: Dynamic Import

// main.js
async function loadModule() {
    const math = await import('./math.js');
    console.log(math.add(2, 3)); // Outputs: 5
}
loadModule();
